## üì¶ Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd educrack
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
